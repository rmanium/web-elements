# Welcome to Sharable Components

<h2>Version {pkg.version}</h2>
This is a starter project for creating sharable components using React.
You can clone the repo and start creating sharable components. For more
information on how to start the project check the repo's [readme](https://github.com/rmanium/react-sharables).

Browse example stories now by navigating to them in the sidebar.
I recommend building UIs with a [**component-driven**](https://componentdriven.org) process starting with [atomic](https://bradfrost.com/blog/post/atomic-web-design/) components and ending with pages.

Please check [here](https://github.com/rmanium/react-sharables) for more
detailed information.
